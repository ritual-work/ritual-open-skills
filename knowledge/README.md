# knowledge/

Where skills persist **reusable knowledge** as [OKF](../docs/knowledge-capture.md)
notes — markdown + YAML frontmatter, one fact per file, readable by any agent.

This folder is **yours**. Skills only ever *offer* to write here, and only with
your approval. Organize by area:

```
knowledge/
├── engineering/   conventions, service relationships, runbooks, risks
├── product/       customer insights, decisions, metric definitions
└── marketing/     positioning, proof points, objections, launch lessons
```

See [example-api-pagination-convention.md](./engineering/example-api-pagination-convention.md)
for the shape, and [docs/knowledge-capture.md](../docs/knowledge-capture.md) for the
rules (optional, always-ask, small + cited, scoped to future agent use).
